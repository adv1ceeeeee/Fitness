# SportWAI — Стратегия монетизации

> Рынок: Россия · Валюта: ₽ · Модель: Freemium + подписка

---

## 1. Что монетизируем прямо сейчас

Всё перечисленное уже реализовано в коде и готово к закрытию за paywall.

### Уровни доступа

```
Free ──────────────────────────── Pro (подписка)
  │                                      │
  ├─ Базовый трекинг подходов            ├─ Полная аналитика
  ├─ До 3 программ тренировок            ├─ Безлимит программ
  ├─ 4 стандартных шаблона               ├─ Схема разминки (NSCA/IPF)
  ├─ История за 30 дней                  ├─ 1ПМ по 4 формулам + %% от 1ПМ
  ├─ Калькулятор блинов (1 вариант)      ├─ 4 варианта блинов (25/20/15/10 кг)
  ├─ Стрик + объём за неделю             ├─ Динамика параметров тела (все метрики)
  └─ Базовые замеры тела                 ├─ Мышечный баланс (пирог)
                                         ├─ Прогресс по упражнению (график)
                                         ├─ История без ограничений
                                         ├─ Экспорт данных (JSON)
                                         └─ Все достижения видны заранее
```

### Почему именно эти ограничения

| Ограничение Free | Логика |
|---|---|
| 3 программы | Новичку хватает 1–2; опытному нужно больше → апгрейд естественен |
| История 30 дней | Прогресс за месяц виден бесплатно, долгосрочный — Pro |
| 1 вариант блинов | Даёт представление о фиче, но профи хочет все варианты |
| Без схемы разминки | Самая «профессиональная» фича — прямой сигнал к апгрейду |

## 3. Дорожная карта роста дохода

### Фаза 1 — Запуск подписки (сейчас → 3 месяца)

**Цель:** первые платящие пользователи, проверка конверсии.

| Задача | Что делать |
|---|---|
| Реализовать `is_pro` в профиле | Поле в `profiles` + провайдер `isProProvider` |
| Paywall-экран | Показывается при попытке открыть Pro-фичу |
| Интеграция платежей | **Rustore** (основной) + **Google Play** (резерв) |
| 7-дневный триал | Убирает барьер входа, конверсия в подписку ~15–25% |
| Промо при регистрации | «Первые 30 дней Pro бесплатно» |

**KPI:** конверсия Free → Pro ≥ 5%

---

### Фаза 2 — Контентный рост (3–9 месяцев)

**Цель:** увеличить ценность Pro, снизить отток.

#### 2a. Готовые Pro-программы (+5–8 новых)
Это чистый контент, не требует новой инфраструктуры:

| Программа | Целевая аудитория |
|---|---|
| PPL (Push/Pull/Legs) 6 дней | Набор массы, опытные |
| 5/3/1 Вендлер | Силовые, 3–4 дня |
| Жиросжигание (HIIT + силовые) | Похудение |
| Турник + брусья (без железа) | Домашние тренировки |
| Женский программа (ягодицы/кор) | Женская аудитория |

#### 2b. Кастомные упражнения → Pro
Код частично готов (`is_standard` в таблице `exercises`). Открыть создание своих упражнений только для Pro.

#### 2c. Расширенная аналитика нагрузки
- Недельный объём по группам мышц
- Сравнение периодов (эта неделя vs прошлая)
- График RPE по сессиям

**KPI:** отток (churn) < 8%/мес

---

### Фаза 3 — Новые потоки дохода (9–18 месяцев)

#### 3a. Тренер в приложении (B2B2C)
Самый высокомаржинальный сегмент рынка:

```
Тренер создаёт программу → назначает клиенту → клиент платит тренеру
SportWAI берёт 20% комиссию
```

| Тариф тренера | Цена | Комиссия SportWAI |
|---|---|---|
| До 5 клиентов | 990 ₽/мес | — |
| До 20 клиентов | 2 490 ₽/мес | — |
| Маркетплейс программ | 15–30% с продажи | ~300–1500 ₽/продажа |

**Почему это важно:** тренер приводит 5–10 клиентов, каждый из которых платит за Pro.

#### 3b. Маркетплейс программ
- Тренеры продают программы за 299–1 990 ₽ (разовая покупка)
- SportWAI берёт 30% → ~90–600 ₽ с каждой продажи
- Не требует подписки от покупателя (альтернативный путь монетизации)

#### 3c. Партнёрские интеграции
- Спортивное питание (реферальные ссылки): +5–15% к заказу
- Спортивные магазины (Decathlon, 2XU, магазины штанг): CPA ~200–500 ₽
- Медицинские лаборатории (анализы крови): реферал за ~500 ₽

## 4. Приоритеты — что делать первым

```
СЕЙЧАС           1–3 мес          3–6 мес          6–12 мес
   │                 │                 │                 │
is_pro поле      RuStore SDK      5 Pro-программ    Тренер-аккаунт
Paywall UI       7-дневный триал  Кастом упражнения Маркетплейс
Ограничения      A/B тест цен     Аналитика нагрузки Партнёрки
Free/Pro         Push-пуш в триал
```

**Главный фокус на ближайшие 6 месяцев:**
1. Запустить подписку (без этого всё остальное бессмысленно)
2. Довести Pro до 8–10 ценных фич (сейчас уже есть 6–7)
3. Добавить 5 Pro-программ (контент продаёт лучше, чем фичи)

## 7. Метрики для отслеживания

| Метрика | Цель | Формула |
|---|---|---|
| Конверсия Free→Pro | ≥ 6% | Pro users / MAU |
| Churn (отток) | ≤ 8%/мес | Отписки / активные Pro |
| LTV | ≥ 1 800 ₽ | ARPU / churn rate |
| CAC | ≤ 200 ₽ | Затраты на привлечение / новые пользователи |
| LTV/CAC | ≥ 3 | LTV / CAC |
| Trial→Paid | ≥ 20% | Перешли из триала / начали триал |
